# Ancient-script-picker-PNSL
Ancient script picker PNSL licensed


Ancient-script-picker-PNSL 1.0.0
Ancient Script Picker (PNSL) is a Windows utility for browsing and copying characters from ancient and historical scripts, including the Phaistos Disc, Linear A and B, Archaic Greek, and Runes. It features tabbed navigation and one-click copy to clipboard. Run the application by simply launching it. The software is released under the Permissive Non-Sale License (PNSL).

Version 1.0.0 was released 2025-10-26.
